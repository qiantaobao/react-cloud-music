依然是，我们做一下redux的数据缓存:

useEffect(() => {
  if(!rankList.length){

另外要提醒大家注意的是，我们将来开发榜单详情页的时候，需要用idx值作为参数来请求，接口文档中已经给出了不同idx值对应的榜单名，但是请求的真实数据中，有些榜单名字不存在接口文档中，也就找不到idx，因此点击榜单的时候我们需要对榜单名做一层校验。

api/utils.js中，加入：

import { RankTypes } from "./config";

//找出排行榜的编号
export const filterIdx = name => {
  for (let key in RankTypes) {
    if (RankTypes[key] === name) return key;
  }
  return null;
};

在Rank/index.js引入:

import { filterIdx } from '../../api/utils';

然后做事件的绑定:

const enterDetail = (name) => {
    const idx = filterIdx(name);
